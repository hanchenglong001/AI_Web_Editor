AI Web Editor Privacy Policy

Last updated: 2026-05-30

Extension version: 2.0.0

Key Point: AI Web Editor is a browser extension that operates entirely within your browser. We do not collect, store, or transmit any personal data to external servers beyond what is necessary to provide the AI editing functionality you request.

1. Data Collection

AI Web Editor collects and processes the following types of information:

1.1 Page Content (Temporary Processing Only)

  • What is collected: HTML, CSS, text content, and computed styles of selected DOM elements on web pages
  • Why it is collected: To enable the AI-powered editing features (translation, shortening, custom styling, etc.)
  • How long it is stored: ONLY while the extension is actively processing your request. Data is NEVER stored permanently or transmitted to third-party servers except through the configured AI model endpoint that you have explicitly selected.

1.2 User Preferences (Stored Locally)

  • What is stored: Theme settings, keyboard shortcut preferences, custom themes, saved snippets
  • Why it is stored: To remember your preferences across browser sessions
  • How it is stored: Exclusively in Chrome's local storage (chrome.storage.local) and sync storage (chrome.storage.sync) on YOUR device. This data NEVER leaves your computer.

1.3 AI Model Communication

  • What is sent: Selected page content and user commands to the configured AI model endpoint (e.g., qwen3.6)
  • Why it is sent: To provide the AI editing functionality
  • Third-party policy: The AI model provider (e.g., Alibaba Cloud / custom backend) may have their own privacy policies that apply to data transmitted through their API. Users are encouraged to review the privacy policy of the configured AI provider.

2. Permissions Used

The extension requires the following Chrome permissions:

  • activeTab — Access to the current tab's DOM to enable element selection and editing
  • contextMenus — Right-click context menu integration
  • storage — Save user preferences and custom data locally
  • scripting — Inject content scripts to enable the editor panel on web pages
  • tabs — Open new tabs for AI editor panel when needed

3. Data Sharing

AI Web Editor does NOT sell, trade, or rent user data to third parties. The only data transmission that occurs is:

  1. Selected page content sent to the configured AI model endpoint for processing (as explained in Section 1.3)
  2. Chrome Web Store may collect anonymous usage statistics as required by Google

4. Children's Privacy

AI Web Editor is not intentionally designed for or directed at children under 13 years of age. We do not knowingly collect personally identifiable information from children under 13.

5. Changes to This Policy

We may update our Privacy Policy from time to time. Any changes will be communicated through the Chrome Web Store listing and within the extension itself. We encourage users to periodically review this page for the latest information on our privacy practices.
