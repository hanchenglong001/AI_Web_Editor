AI Web Editor CWS Screenshots Directory
========================================

Required for Chrome Web Store publication.

Screenshots needed:
1. screenshot-main.png        (1280x800) — Main editor interface showing the floating panel with quick commands
2. screenshot-context-menu.png (1280x800) — Right-click context menu with AI editing options  
3. screenshot-translate.png    (1280x800) — Multi-language translate with auto-detection
4. screenshot-theme.png        (1280x800) — Custom theme editor interface
5. screenshot-shortcut.png     (1280x800) — Keyboard shortcut manager in popup

Instructions:
- Take screenshots from the actual extension running in Chrome
- Or use Chromium's "Capture Node" in DevTools to render preview.html
- Recommended format: PNG, min 320px, max 1280px (WIDESCREEN_16x9 preferred)
- Must be .png or .jpg/jpeg files

To generate from preview.html:
1. Open screenshots/preview.html in Chrome
2. Use DevTools -> Screenshot -> Capture full size screenshot for each section
3. Save as PNG in this directory
